# excel-dashboard-for-coffee-shop-sales
This project is an interactive Excel dashboard created to analyze coffee shop sales performance using transactional data.
The dashboard helps understand sales trends, customer behavior, product performance, and time-based patterns in a simple and visual way.

It is designed for data analysis practice, portfolio showcase, and business insights.
